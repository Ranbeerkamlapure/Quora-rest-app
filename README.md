# Quora REST App

A simple RESTful CRUD application built with Node.js, Express, and EJS.

#Live Demo
https://quora-rest-app.onrender.com/

#Features
- Create, Read, Update, Delete posts
- RESTful routing
- Method Override (PATCH & DELETE)
- Deployed on Render

#Tech Stack
- Node.js
- Express.js
- EJS
- UUID
- Method-Override

#Note
This project uses in-memory storage. Data resets when the server restarts.
